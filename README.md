# 经颅光刺激仿真评估工作站

这是一个用于 GitHub Pages 发布的静态 HTML 原型，包含登录页、七步式工作站内页和治疗方案报告 PDF 下载。

## 页面入口

- 登录页：`index.html`
- 工作站内页：`ui-static/index.html`
- 报告 PDF：`output/pdf/经颅光刺激治疗方案报告.pdf`

## 功能流程

1. 项目中心
2. 影像分析
3. 靶点规划
4. 参数设置
5. 仿真评估
6. 方案确认
7. 报告输出

## GitHub Pages

仓库已包含 `.github/workflows/pages.yml`。推送到 `main` 分支后，GitHub Actions 会将静态站点发布到 GitHub Pages。
